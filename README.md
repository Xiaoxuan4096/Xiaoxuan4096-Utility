# Xiaoxuan4096-Utility
自己编写的一些实用程序。

---
## 内容
### BasicSupport
这是对其他库的基本支持。
- FileBase : 基础文件操作函数。
- StringUtilities : 字符串操作函数。
- ObjectUniqueIdentityString : 对象唯一标识符。通过随机数产生。
- Exception : 异常处理类。
### FileEditor
基础的文件读写库。
